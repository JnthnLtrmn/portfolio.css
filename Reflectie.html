Reflectie

Sterke punten

  • Analyseren: Een van mijn sterkpunten is het analyseren van code en stapsgewijs bestuderen wat wanneer gebeurt. Zo bekijk ik ook hoe is opgemaakt is en kan hier op uit maken hoe iets in elkaar gestoken is.
  • Leergierig: Iets bij leren in een steeds veranderende wereld is een groot plusppunt wat ik mee draag. Iets bijleren is een motivatie wat mij stimuleert. Ook om bestaande kennis bij te schaven geeft mij een motiviatie.
  • Structuur: Structuur is voor mij een manier van logisch te werk gaan. Door gestructureerd te werk te gaan behoud je het overzicht van waar je mee bezig bent. Als er een probleem voor komt kan er ook gemakkelijker gezocht worden naar een oplossing voor het probleem.

Werkpunten

  • Zelfeffectief: Ik schat mijzelf fout in waarbij het resultaat meestal beter uitkomt als wat ik zelf gedacht had. het resultaat kan ook fout zijn maar hier leer ik dan uit door te kijken waar ik volgende keer op moet letten.
  • Stuurloos: Omdat ik het best stapsgewijs te werk ga, wordt het voor mij moeilijk om te bepalen hoe te beginnen aan een project. Hier voor is mijn kennis over het programeren nog beperkt maar dit gaat verbeteren door hier meer mee te werk gaan.
  • Concretiseren: Door mijn beperkte kennis weet ik niet altijd wat waar een wanneer gebruikt moet worden. Hierdoor kan ik wel eens vast komen te lopen waardoor er tijd verloren gaat naar opzoek werk.

POP sessie: Personal branding

Bij deze sessie leerden we hoe belangrijk het is om profesioneel over te komen via sociale media kanalen. Er werden duidelijke verschillen gegeven wat een profesioneel 'visite kaartje' maakt en welke gegevens hierin voor komen. Bij dze sessie hebben we een Linkd-in account gemaakt en je zag hier snel hoe belangrijk dit platform is om connecties te maken binnen de IT wereld. Met alle do's en don'ts werden we voorzien met de kennis die nodig is om ons later in het werkveld de juiste eerste indruk kan geven.
