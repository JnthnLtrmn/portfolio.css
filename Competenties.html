Competenties

Ontwerper

Als ontwerper kan ik mijn creativiteit aanspreken. Aan de hand van een gekregen opdracht kan ik redelijk goed inschatten wat de nodige tijd is om het doel te bereiken. Momenteel is mijn kennis nog beperkt tot HTML voor websites te ontwerpen en voor het ontwerpen met het .NET framework heb ik de XAML ook al goed onder de knie. Indien er technische uitdagen voorkomen zoek ik hierover info op via het web of overleg ik met collega's wat de mogelijkheden zijn om dit uit te voeren. Voor mijn eigen heb ik het gevoel dat ik als ontwerper goed op weg ben om hier profesioneel mee aan de slag te kunnen gaan. Redelijk snel had ik door dat ontwerpen wel iets is wat mij ligt.

Programmeur

Programmeren is een geweldig onderdeel van de IT wereld. Hierin is mijn kennis nog niet groot genoeg om een opdracht volledig af te werken. Aan de hand van een opdracht weet ik wat er moet gebeuren en welke richting er uit gegaan moet worden. Maar uit eigen ervaring voel ik ook dat ik hier nog veel werk voor de boeg heb. Een groot deel van een code kan ik zelfstandig schrijven. Maar tot nu toe bots ik nog vaak op de eindafwerking van een code. Hiervoor ben ik nog veel aan het oefenen en ben ik geleidelijk aan toch aan het groeien naar een volwaardige programmeur.

Tester

Het testen van mijn projecten is een heel handig gegeven als ik aan een project bezig ben. Ik heb geleerd dat als ik met een project bezig ben dit kan helpen om verder te geraken. Vooral wanneer ik bezig ben met het programmeren van een applicatie heeft mij dit al vaak geholpen wanneer ik ergens vast bij kwam te zitten. Als tester zit ik nog met een basiskennis die ik nog moet veruimen door meer te weten komen welke tools er nog meer zijn.

Teamspeler

Een heel belangrijk gegeven aan een project is de communicatie tussen de personen waarmee je samenwerkt. Dit is voor mij een heel belangrijk gegeven omdat je elkaar kan helpen en zo kan leren van elkaar. Gaandeweg mijn traject heb ik gemerkt dat ik hier sterk in ben doordat wanneer er een vraag gesteld wordt, ik alles in het werk zal stellen om het probleem op te lossen. Een punt waar ik aan moet werken is zelf ook vragen durven stellen en niet met het gedacht zitten dat ik hiermee voor overlast zorg.

Levenslang leren binnen de IT

In de loop van mijn traject heb ik gemerkt hoe snel de IT wereld kan veranderen. Toen ik begon aan mijn opleidingstraject dacht ik al redelijk wat kennis te hebben van IT, maar in mijn opleiding heb ik nog veel geleerd. Door met mijn opleiding bezig te zijn heb ik meer opgelet wat er allemaal in de IT wereld aan het gebeuren was en ik blijf dit ook opvolgen zodat ik de evolutie hiervan bij kan blijven benen om later eventueel ook een steentje bij te kunnen dragen aan de evolutie van de IT sector.
